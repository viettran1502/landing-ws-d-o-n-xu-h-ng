# Workshop Xu hướng Video ngắn & Bán hàng 2026

Landing page for Duy Muối's workshop on short-form video trends.

## Quick Start
```bash
# Open in browser
start index.html

# Or serve with any static server
npx serve .
```

## Deploy
Push to GitHub → Connect to Cloudflare Pages → Done

## Tech
- HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- Single file, no build

## Customization
1. Replace `#payment` with actual payment link
2. Add speaker photo (replace placeholder)
3. Update dates if needed
