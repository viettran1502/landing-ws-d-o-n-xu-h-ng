# Workshop Landing Page - Dự đoán Xu hướng Video ngắn & Bán hàng 2026

Vietnamese landing page for Duy Muối's workshop on short-form video trends and sales strategies for 2026.

## Quick Start

```bash
# Open in browser
start index.html

# Or serve locally
npx serve .
```

## Tech Stack
- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript
- Single file: `index.html`
- No build step required

## Deploy to Cloudflare Pages
1. Push to GitHub
2. Connect repo to Cloudflare Pages
3. Build command: (none)
4. Output directory: `/`

## File Structure
```
├── index.html          # Main landing page
├── assets/             # Images/media (placeholder)
├── docs/               # Documentation
└── plans/              # Implementation plans
```

## Features
- Responsive design (mobile-first)
- Countdown timers (workshop + early bird)
- Count-up statistics animation
- Accordion sections (FAQ, Curriculum)
- Smooth scroll navigation
- Fade-in scroll animations
- Mobile sticky CTA bar

## Customization
- Replace `#payment` links with actual payment URL
- Add speaker photo (300x300) to replace placeholder
- Update countdown dates if workshop is rescheduled
